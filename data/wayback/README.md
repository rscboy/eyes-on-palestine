# Wayback Archive Status

`archive_status.json` is generated by the GitHub Actions workflow `Wayback Archive Articles`.

Do not put Internet Archive credentials in this directory or anywhere in the website. The workflow reads them from GitHub Secrets:

- `IA_S3_ACCESS_KEY`
- `IA_S3_SECRET_KEY`

The archiver submits article source URLs to the Internet Archive Save Page Now endpoint in batches of 6, waiting 65 seconds between batches.
